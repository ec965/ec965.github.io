HC2: IoT Enviorment Sensor

This is an IoT platform for the Rotronics HC2 humidity and temperature probe. Using Raspberry Pi and Arduino, I am devloping communication over WiFi, SMS, Email, and LoRa.

I am creating an Arduino Library and R-Pi Python API for the HC2.

Because this is a consulting project, all my code is private. Please contact me at enoch965@gmail.com if you are interested in more information on this project.
