HC2: IoT Enviorment Sensor

August 2019 - November 2019

GitHub

I am creating an IoT platform for the Rotronics HC2 humidity and temperature probe. Using Raspberry Pi and Arduino, I am devloping communication over WiFi, SMS, Email, and LoRa. Temperature data is sent using Low-power radio from a Moteino (Arduino Uno + RFM95 LoRa chip) to another Moteino that is connected to a Raspberry Pi. Warnings can then be triggered if the temperature is too high or too low. These warnings can be sent out over Email using the SMTP Python API.

I have created an Arduino Library and Raspberry Pi Python API for interfacing with the HC2.
