GPS Sound Sensor (GPSSS)

September 2019 - March 2020

Project Site GitHub

For our senior design project, we decided to create a sensor that can detect sounds and transmit thead that data to an android application. We used a GPS to get the location and a compass to get the heading of the device. An ESP8266 was used for WiFi connectivity.

Initially we experimented with using a Beagle Bone Black but settled on using a Teensy 4.0 as the microcontroller.

My main contribution was interfacing the GPS and compass as well as setting up the MQTT on both device and android application side.

Special thanks to Minh-Tam Chau, Sam Longo, and Kelwin Lee.
