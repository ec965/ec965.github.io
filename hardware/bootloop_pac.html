Automatic Bootloop Stress Tester

Summer 2019

While interning for Panasonic Avionics, I frequnetly had to do boot loop stress tests on LRUs (Line-Replaceable Unit). Imagining sitting there and staring at a serial monitor and then flipping a switch when it finally booted. After awhile, my manager gave me a raspberry pi and an SSR so I could automate this job. I wrote a simple Python script that waits for the Serial monitor to get to a certain boot position, logs into the LRU, then reboots using the SSR. I wonder how much money I saved them.
