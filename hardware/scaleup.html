Scale Up

SD Hacks 2019

DevPost GitHub

Scale Up is a smart food scale that can calculate the calories of whole foods based on weight. Caloric data is tracked in the iOS health app and a Google Sheet using IFTTT. Threaded python firmware keeps buttons responsive and preserves functionaility of a traditional food scale. USDA Nutrition API is used to find caloric data of various food types. Check out the DevPost for my inspiration and more information!
